Создать класс Reader, который будет хранить информацию о пользователе библиотеки,
включая ФИО, номер читательского билета, факультет, дату рождения и телефон.
Разработать программу, в которой создается массив 3 объектов данного класса
и вызываются по очередно оба метода, для каждого из экземпляров класса.

Перегрузить методы takeBook() и returnBook() в классе Reader:

Метод takeBook(), который принимает количество взятых книг и выводит на консоль
сообщение вида "Петров В. В. взял X книги", где X - количество взятых книг.
Метод takeBook(), который принимает переменное количество названий книг
и выводит на консоль сообщение вида
"Петров В. В. взял книги: Приключения, Словарь, Энциклопедия",
где перечислены названия взятых книг.

Метод takeBook(), который принимает переменное количество объектов класса Book
(где Book - новый класс, содержащий имя и автора книги) и выводит на консоль сообщение
вида "Петров В. В. взял книги: Приключения, Словарь, Энциклопедия",
где перечислены названия и авторы взятых книг.

Аналогично перегрузить метод returnBook() в классе Reader:

Метод returnBook(), который принимает количество возвращенных книг и выводит на консоль
сообщение вида "Петров В. В. вернул X книги", где X - количество возвращенных книг.
Метод returnBook(), который принимает переменное количество названий книг и выводит
на консоль сообщение вида "Петров В. В. вернул книги: Приключения, Словарь, Энциклопедия",
где перечислены названия возвращенных книг.
Метод returnBook(), который принимает переменное количество объектов класса Book
и выводит на консоль сообщение вида "Петров В. В. вернул книги: Приключения, Словарь, Энциклопедия",
где перечислены названия и авторы возвращенных книг.
